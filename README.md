<h1 align="center">Mybricks</h1>

<p align="center">Mybricks 组件库开发插件</p>

## 快速开始

- 点击 VS Code 活动栏上的「插件商店图标」；
- 在输入框中输入「Mybricks」进行搜索；
- 安装成功后，点击 VS Code 界面左侧活动栏上的「Mybricks 图标」，开始使用。
